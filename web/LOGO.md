# Matcher Logo Guidelines

## Logo Design Requirements

The Matcher logo should be a simple heart symbol with:
- Slight tilt to show connection
- Rounded edges
- One-color version available for dark and light modes

## Required Assets

Create the following PWA icon files:

1. **pwa-192x192.png** - 192x192px PNG
2. **pwa-512x512.png** - 512x512px PNG

## Logo Variations

- **Full-color**: Crimson Pulse (#C62828) on transparent background
- **White version**: For dark backgrounds
- **Black version**: For light backgrounds

## Clear Space

Minimum padding around logo: width of the heart shape

## Use Cases

- App icon
- Splash screen
- Loading animation
- Match screen animation

## Design Tools

You can create the logo using:
- Figma
- Adobe Illustrator
- Canva
- Or any vector graphics tool

Export as PNG at the required sizes for PWA icons.
